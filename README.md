## Getting Started
### Prerequisites

You need to install Gradle
### Installation

1 Clone the repo
   ```sh
   git clone https://github.com/crudl4d/graphbook-UI.git
   ```
2 Build the project, either with Gradle or IDE
   ```sh
   gradle build
   ```
